15EJEMPLOS DE ETIQUETAS O (TAGS) MAS CRUCIALES EN MICROSOSFT VISUAL STUDIO

1) Doctype Esta etiqueta es muy importante es la que le dice al ordenador y al navegador que tipo de HTML estas usando

2) Tittle es el que define el titulo que aparece en la pestaña del navegador

3) Body se usa para contener todo el contenido dentro de el, como este ejemplo de etiquetas de no usar la etiqueta llevaria a problemas de incompatibilidad en ciertos navegadores

4) Head se usa para gregar "metadatos" datos sobre datos que no vemos directamente en la pagina pero que son utiles para ayudar a los navegadores a entender mejor los datos

5) H1 AL H6 lo usamos para crear titulos y encabezados de mayor h1 a menor h6

6) Link se usa para conectar cosas externas a tu página web. ejemplo, si tienes un archivo con el diseño de tu página (como colores, fuentes y márgenes)

7) header se usa para definir la parte superior de una página web o de una sección específica de ella

8) footer Define la parte inferior de la página, con información como derechos de autor o contactos. Es como la última página del libro con información de contacto o agradecimientos.

9) nav Contiene enlaces de navegación para moverte por la página o el sitio web. Es como el índice en un libro que te dice dónde encontrar cosas.

10) main Enmarca el contenido principal de la página, separado del encabezado y el pie de página. Es como el capítulo principal del libro.

11) p Define un párrafo de texto. Es como un bloque de texto en un libro, separado por espacios de otros bloques.

12) a Crea un enlace a otra página o a una parte de la misma página. Es como un marcador en un libro que te lleva a otra página o sección.

13) section Agrupa contenido relacionado en secciones. Es como un capítulo o una sección del libro que trata un tema específico.

14) ul Define una lista desordenada con viñetas. Es como una lista de puntos en un libro sin un orden específico.

15) ol Define una lista ordenada con números. Es como una lista numerada en un libro con un orden específico.
